# Network Watch 개인정보처리방침

**최종 업데이트: 2026년 2월 4일**

---

## 1. 개요

Network Watch(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 본 개인정보처리방침은 앱 사용 시 수집되는 정보와 그 처리 방법에 대해 설명합니다.

---

## 2. 수집하는 정보

### 2.1 앱에서 직접 수집하는 정보

**Network Watch는 사용자의 개인정보를 직접 수집하지 않습니다.**

앱에서 입력하는 모든 데이터(웹사이트 URL, IP 주소, MAC 주소 등)는 기기 내부에만 저장되며 외부 서버로 전송되지 않습니다.

### 2.2 iCloud 동기화

사용자가 iCloud 동기화를 활성화한 경우, 앱 데이터는 사용자의 개인 iCloud 계정에 저장됩니다. 이 데이터는 Apple의 개인정보처리방침에 따라 관리됩니다.

### 2.3 제3자 서비스를 통해 수집되는 정보

#### Google AdMob (광고)

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

- 기기 식별자 (광고 ID/IDFA)
- IP 주소
- 기기 정보 (모델, OS 버전)
- 광고 상호작용 데이터

자세한 내용은 [Google 개인정보처리방침](https://policies.google.com/privacy)을 참조하세요.

#### 앱 추적 투명성 (ATT)

iOS 14.5 이상에서는 앱이 사용자 추적 전 동의를 요청합니다. 사용자는 이를 거부할 수 있으며, 거부 시 맞춤형 광고 대신 일반 광고가 표시됩니다.

#### IP 위치 조회 서비스

IP 위치 조회 기능 사용 시 ipwho.is 서비스에 조회하려는 IP 주소가 전송됩니다. 이는 사용자의 개인 IP가 아닌, 사용자가 직접 입력한 IP 주소입니다.

---

## 3. 인앱 구매

프리미엄 구매는 Apple App Store를 통해 처리됩니다. 결제 정보는 Apple에서 직접 관리하며, 앱은 구매 상태(구매 여부)만 확인합니다. 결제 정보는 앱에 저장되지 않습니다.

---

## 4. 데이터 저장 위치

| 데이터 유형 | 저장 위치 | 공유 여부 |
|------------|----------|----------|
| 웹사이트 목록 | 기기 내부 / iCloud | 공유 안함 |
| 모니터링 기록 | 기기 내부 / iCloud | 공유 안함 |
| 설정 | 기기 내부 | 공유 안함 |
| 광고 데이터 | Google 서버 | Google 정책에 따름 |

---

## 5. 데이터 보안

- 모든 네트워크 통신은 HTTPS를 통해 암호화됩니다
- 앱 데이터는 iOS 보안 메커니즘에 의해 보호됩니다
- iCloud 데이터는 Apple의 암호화 표준으로 보호됩니다

---

## 6. 사용자 권리

### 6.1 데이터 삭제

앱 내 설정 > 데이터 관리 > 모든 데이터 삭제 기능을 통해 모든 앱 데이터를 삭제할 수 있습니다.

### 6.2 앱 삭제

앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다. iCloud에 저장된 데이터는 iOS 설정에서 별도로 삭제할 수 있습니다.

### 6.3 광고 추적 관리

- iOS 설정 > 개인정보 보호 > 추적에서 앱 추적을 관리할 수 있습니다
- iOS 설정 > 개인정보 보호 > Apple 광고에서 맞춤형 광고를 관리할 수 있습니다
